Manual Test: Add a popcorn event

This test makes sure that popcorn events can be added without incident.

Steps to Follow:

  1. Wait for page to load, make sure HTML5 video is displayed on page
  2. Drag the "titles" plugin from the +Popcorn Editor over the video
  3. In the new Editor that appears enter some text in the text input element
  4. Click "done" to create the event
  5. Drag the "image" plugin from the +Popcorn Editor over the video
  6. In the new Editor that appears, press the radio button beside "URL"
  7. In the new Editor that appears, paste this URL http://mozillapopcorn.org/wp-content/uploads/2011/11/icon_popcorn_js-150x150.png into the source URL field
  8. Scroll the playhead to a time before the events start
  9. Play through

Expected Result

The text and image should appear over the video at the specified time.
